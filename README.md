# HikeHack
An ACV Hackathon project.

HikeHack aims to help discover hikes, and view your previous hikes.

## Backend

### Project setup
```
cd server
go run main.go
```
## Frontend
### Project setup
```
yarn install
```

#### Compiles and hot-reloads for development
```
yarn serve
```

#### Compiles and minifies for production
```
yarn build
```

#### Lints and fixes files
```
yarn lint
```
